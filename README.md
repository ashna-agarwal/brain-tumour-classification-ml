# brain-tumour-classification-ml
Brain Tumor Detection using Transfer Learning with MobileNetV2 and Grad-CAM explainability. This project classifies MRI scans into Glioma, Meningioma, Pituitary Tumor, and No Tumor categories, achieving over 90% accuracy through deep learning and providing interpretable predictions using Grad-CAM visualizations. 

# Brain Tumor Detection using MobileNetV2

A deep learning project for multi-class brain tumor classification from MRI scans using MobileNetV2 transfer learning. The model classifies MRI images into:

- Glioma
- Meningioma
- Pituitary Tumor
- No Tumor

## Features
- MobileNetV2 Transfer Learning
- MRI Image Classification
- Exploratory Data Analysis (EDA)
- Confusion Matrix & Classification Report
- Training & Validation Curves
- Grad-CAM Explainability
- Comparative Literature Review

## Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Pandas
- Matplotlib
- Seaborn

## Results
- >90% Classification Accuracy
- Multi-class MRI Classification
- Explainable AI through Grad-CAM visualizations
